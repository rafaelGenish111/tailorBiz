# Skill: Frontend Specialist

Use this skill when handling UI/UX tasks, React components, or CSS styling.

## Capabilities
- Implements responsive designs.
- Uses modern React patterns (Hooks, Context).
- Strictly follows the design system tokens.

## Process
1. Check existing components in `@src/components` before creating new ones.
2. If a backend API is missing, create a mock interface and ask the Orchestrator to trigger the Backend Skill.
