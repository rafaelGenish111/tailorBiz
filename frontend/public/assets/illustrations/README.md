# הוראות להורדת איורים

## איורים נדרשים

יש להוריד איורים חינמיים מהאתרים הבאים ולשמור אותם בשמות הבאים:

1. **hero-illustration.svg** - איור למערכת אוטומציה/ניהול עסקי
   - אתרים מומלצים:
     - [undraw.co](https://undraw.co/illustrations) - חפשו "automation", "business", "dashboard"
     - [storyset.com](https://storyset.com) - חפשו "business", "automation"
     - [icons8.com/illustrations](https://icons8.com/illustrations) - חפשו "business automation"

2. **features-illustration.svg** - איור לתכונות ושירותים
   - אתרים מומלצים:
     - [undraw.co](https://undraw.co/illustrations) - חפשו "features", "services"
     - [storyset.com](https://storyset.com) - חפשו "features", "services"

3. **automation-illustration.svg** - איור לאוטומציה ותהליכים
   - אתרים מומלצים:
     - [undraw.co](https://undraw.co/illustrations) - חפשו "automation", "workflow"
     - [icons8.com/illustrations](https://icons8.com/illustrations) - חפשו "automation"

## הוראות

1. היכנסו לאחד האתרים המומלצים
2. חפשו איור מתאים לפי המילות מפתח
3. הורידו את האיור בפורמט SVG
4. שמרו את הקובץ בשם המתאים בתיקייה זו (public/assets/illustrations/)
5. החלפו את קבצי ה-placeholder הקיימים

## הערות

- הקבצים הנוכחיים הם placeholder בלבד
- ודאו שהקבצים בפורמט SVG
- הקבצים צריכים להיות מותאמים לנושא TailorBiz (ניהול עסקי מותאם אישית, אוטומציה)
